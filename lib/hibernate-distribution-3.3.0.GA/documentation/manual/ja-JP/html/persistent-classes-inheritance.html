4.2. 継承の実装

サブクラスも1番目と2番目のルールを守らなければなりません。 サブクラスはスーパークラス Cat から識別子プロパティを継承します。 

package eg;

public class DomesticCat extends Cat {
        private String name;

        public String getName() {
                return name;
        }
        protected void setName(String name) {
                this.name=name;
        }
}
